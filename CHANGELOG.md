# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-09
### Added
- Initial public release of ASTRA
- Vedic astrology calculation backend (FastAPI)
- Next.js cinematic frontend with Framer Motion animations
- AI interpretation integration using OpenRouter
- Geocoding and Timezone resolution via Nominatim
- Fully responsive design with mobile focus support
- Accessible ARIA compliance and prefers-reduced-motion support
