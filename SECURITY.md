# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report any security vulnerabilities by opening a private security advisory on GitHub or emailing security@example.com (placeholder) rather than creating a public GitHub issue.

If the vulnerability is confirmed, we will release a patch as quickly as possible and coordinate the disclosure with you.
